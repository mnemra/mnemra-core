# mnemra-core

mnemra-core is the core context layer for Mnemra — an MCP-native system for persistent, structured memory that integrates with AI development workflows.

This documentation covers the internal architecture, design decisions, and developer guides for mnemra-core contributors and integrators.

## About This Site

This docs site is built with [mdBook](https://rust-lang.github.io/mdBook/) and includes:

- **Architecture Decision Records (ADRs)** — the reasoning behind key design choices
- **Developer Guides** — how to build, extend, and contribute
